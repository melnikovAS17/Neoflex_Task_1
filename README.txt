Для получения расчёта отпускных без учёта праздников :
    localhost:8080/calculacte/{среднегодовая зп}/{кол-во дней отпуска}
С учётом праздничных дней :
    localhost:8080/accurateCalculacte/{среднегодовая зп}/{кол-во дней отпуска}?dayOfLeaving={значение}&monthOfLeaving={значение}

<<<
    Для расчёта с учётом использовал субд PostgreSQL, создал одну таблицу с информацией о месяцах, а именно:
    month_number - номер меясца ( 1 - 12 ),
    holidays - государственные праздники, которые не будут включаться в оплачиваемые дни отпуска
    quantity_days - количестов дней в месяце (29-31, брал за основу 2024 год)
>>>

Структура таблицы в бд

-------------------------------------------------------
|  id     |month_number|    holidays   | quantity_days|
-------------------------------------------------------
|  1     |      1     |1,2,3,4,5,6,7,8|       31      |
-------------------------------------------------------
|  2     |      2     |       23      |       29      |
-------------------------------------------------------
|  ...   |    ...     |     ...       |   ...         |
-------------------------------------------------------

SQl :

create table month_info
      (
          id            integer generated by default as identity,
          month_number  integer            not null
              constraint weekends_inmonth_month_number_key
                  unique
              constraint weekends_inmonth_month_number_check
                  check ((month_number < 13) AND (month_number > 0)),
          weekends      varchar,
          quantity_days integer default 28 not null
      );
